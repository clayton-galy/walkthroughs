Napping: 1.0.1

April 14, 2022

Napping features a Ubuntu web server with a simple login page and a way to register a new user. I’ll create a new user and login to the website. Once logged in, I am able to send a link to an administrative user who will automatically click on my link. With this in mind, I’ll take advantage of the tabnabbing vulnerability which will allow me to phish the administrative user’s credentials. I’ll use the credentials to login via SSH as the “daniel” user who has write access to a file that’s being executed by a cronjob. I’ll take advantage of this to get a shell as the “adrian” user. The adrian user has sudo privileges to execute vim as the root user. I’ll use this to get a shell as the root user.

Empire: LupinOne

April 12, 2022

LupinOne is a Ubuntu web server running a simple website. I’ll use wfuzz to fuzz for files, finding a hint towards hidden files on the web server. I’ll create a custom wordlist to fuzz the site further to eventually find an encoded, encrypted private SSH key. Once I crack the password for the key, I’ll login to the box via SSH as the “icex64” user. The icex64 user can run a python script as another user on the box. I’ll pollute a library the script is using to get a shell as the arsene user. The arsene user can run the pip command as root, and I’ll use this to get a shell as the root user.

Empire: Breakout

April 12, 2022

Breakout is a Debian web server running an instance of Webmin. There is a comment in the HTML of the website that contains an encoded password. I’ll decode this password to login to the Webmin site. Once logged in, I’ll create a cronjob to execute a reverse shell to get a shell on the box as the cyber user. On the box, the tar command has special capabilities set to read all files. I’ll use this to my advantage by reading a backup file I don’t usually have read access to to get the root user’s cleartext password.
